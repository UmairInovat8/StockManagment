# Toolset: Audit Diagnostic Toolbox

This branch contains a collection of diagnostic and utility scripts for managing the audit lifecycle and verifying system state.

## Key Scripts:
- `find-auditor-tasks.js`: Diagnostic script to list active assignments for a specific auditor.
- `verify-assignments-again.js`: Detailed verification of location codes and database mapping.
- `assign-sections.js`: Utility to programmatically assign audit sections for testing.
- `mock-api-response.js`: Tool to simulate and verify API payload structures.
- `diagnose-audit.ts`: Full-stack diagnostic for identifying stuck or mismatched audit sessions.
